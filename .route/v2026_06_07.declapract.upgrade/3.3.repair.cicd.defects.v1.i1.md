## defects fixed

### defect 1: keyrack extends paths not found

**what**: CI failed with `BadRequestError: extended keyrack not found`

**how** (diff): declapract added extends to `.agent/keyrack.yml`:
```yaml
extends:
  - .agent/repo=bhrain/role=reviewer/keyrack.yml
  - .agent/repo=bhuild/role=dispatcher/keyrack.yml
  - .agent/repo=ehmpathy/role=mechanic/keyrack.yml
```

but these files don't exist in this repo.

**why**: declapract template assumes all ehmpathy repos have the full agent directory structure. this repo was missing the `rhachet-roles-rhachet` package which provides the symlinked roles for keyrack lookup.

**fix**: added `rhachet-roles-rhachet` to `package.json` and updated `prepare:rhachet` command to include all required roles (dispatcher, dreamer). keyrack extends now lookup via rhachet-roles packages.

### defect 2: depcheck flagged rhachet-roles-rhachet as unused

**what**: lint failed with `Unused devDependencies: rhachet-roles-rhachet`

**how** (diff): the new package was added but depcheck doesn't know it's used by rhachet init.

**why**: depcheck can't trace usage through CLI commands like `prepare:rhachet`.

**fix**: added `rhachet-roles-rhachet` to `.depcheckrc.yml` ignores list.
