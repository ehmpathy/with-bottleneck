## defects fixed

### defect 1: keyrack extends paths not found

**what**: CI failed with `BadRequestError: extended keyrack not found`

**how** (diff): declapract added extends to `.agent/keyrack.yml`:
```yaml
extends:
  - .agent/repo=bhrain/role=reviewer/keyrack.yml
  - .agent/repo=bhuild/role=dispatcher/keyrack.yml
  - .agent/repo=ehmpathy/role=mechanic/keyrack.yml
```

but these files don't exist in this repo.

**why**: declapract template assumes all ehmpathy repos have the full agent directory structure with bhrain and bhuild roles. this small utility repo only has the mechanic role from ehmpathy, not the reviewer or dispatcher roles from bhrain/bhuild.

**fix**: removed the extends array from `.agent/keyrack.yml` since the extended files don't exist.
